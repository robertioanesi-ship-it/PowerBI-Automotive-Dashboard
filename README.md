# 🏭 Automotive Manufacturing Dashboard — Power BI

## Overview
Interactive Power BI dashboard monitoring key manufacturing KPIs 
in the automotive industry, built using Power BI and MySQL.

## Dashboard Pages
- **Production Overview** — Total Units Produced (2.25M), OEE, 
Scrap Rate and Total Downtime tracked monthly
- **Scrap Analysis** — Scrap Rate %, Total Units Scrapped (63K) 
and Scrap Rate PM monitored across production lines
- **OEE Analysis** — OEE (91.72%), Availability (94.36%), 
Performance (74.82%) and Quality (97.20%) breakdown
- **Key Influencers (AI Visual)** — AI-powered analysis identifying 
Downtime_Reason as the main driver of performance decrease

## Tools Used
- Power BI (DAX, data modeling, AI visuals)
- MySQL (data extraction)
- Excel (data preparation)
