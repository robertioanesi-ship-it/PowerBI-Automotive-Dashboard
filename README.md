# 🏭 Automotive Manufacturing Dashboard — Power BI

## Overview
Interactive Power BI dashboard tracking key manufacturing KPIs 
in the automotive industry.

## Metrics Tracked
- OEE (Overall Equipment Effectiveness)
- Production Performance
- Scrap Rates

## Tools Used
- Power BI (DAX, data modeling)
- MySQL (data extraction)
- Excel (data cleaning)
